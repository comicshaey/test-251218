업무 오류·누락 방지 체크리스트

자주 실수하거나 누락되기 쉬운 행정업무 절차를 실제 업무 흐름 기준으로 점검할 수 있도록 정리한 체크리스트입니다.
실제 업무 상황에 맞게 수정·보완하면서 사용하는 것을 전제로 합니다.

예시) 계약제 교원 인건비 처리 전 확인

  • 임용 기간, 근무 형태(상시/시간제) 확인
  • 단체협약 및 근로계약서 상 임금 조건 확인
  • 나이스 인사·급여 입력 완료 여부 확인
  • 사회보험 자격 취득·상실 신고 처리 여부 확인
  • 예산 배정 내 집행 가능 금액 검토

예시) 교육공무직 연차·근태 관련 확인

  • 연차 발생 기준일과 기준기간 재확인
  • 출근율 산정 시 결근·무급휴직 반영 여부
  • 업무 단위별 근무일수 집계 검증
  • 연차 사용 내역과 미사용 연차 산정 기준 일치 여부
